# 💰 API de Controle de Gastos

Projeto de uma API REST para gerenciamento de despesas, com dashboard interativo para visualização de dados.

## 🚀 Funcionalidades

- Cadastro de gastos
- Listagem de gastos
- Edição de gastos
- Exclusão individual e total
- Filtro por categoria
- Ordenação de dados
- Validação de formulário
- Dashboard com métricas
- Gráfico de despesas (pizza e barras)

## 📊 Dashboard

O sistema possui um frontend simples com:

- Visualização dos gastos
- Total de despesas
- Maior gasto
- Gasto médio
- Número de registros
- Gráficos interativos

## 🛠️ Tecnologias utilizadas

- Python
- FastAPI
- Uvicorn
- HTML
- CSS
- JavaScript
- Chart.js
- JSON (armazenamento de dados)
